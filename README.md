# Walrus Skills

Walrus Skills is a decentralized skills marketplace and developer hub to help builders ship faster on **Walrus** and the **Sui** ecosystem.

It lets developers discover, share, and integrate reusable:
- skills & workflows
- AI prompts
- SDK examples & APIs
- templates
- infrastructure modules

The UI is themed to match `public/landing.png` (mint palette, light grid background).

## Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev` – start the dev server
- `npm run build` – production build (webpack mode)
- `npm run start` – start the production server
- `npm run lint` – run ESLint

## Project structure

- `src/app` – App Router pages/routes
- `src/components` – UI + feature components
- `public/landing.png` – landing banner image
