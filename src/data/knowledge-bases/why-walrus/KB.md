# What is Walrus?

Walrus is a decentralized programmable storage protocol built for the Sui ecosystem. It is designed to store large binary objects (blobs) such as AI datasets, videos, images, NFTs, documents, application assets, and agent memory at scale.

Walrus combines:
- decentralized storage
- high availability
- programmable data ownership
- low-cost blob storage
- Sui-native composability

Unlike traditional blockchain storage, Walrus stores data efficiently off-chain while coordinating ownership, access, and verification through Sui.
